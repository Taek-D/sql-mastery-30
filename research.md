  - 비디오 해설 제공

### 3) **Mode Analytics SQL Tutorial**
- **핵심 강점**:
  - 실무 케이스 기반 튜토리얼

  - 비즈니스 맥락이 있는 문제
- **한계**: 성능 최적화 내용 부족, 기초 위주

### 4) **BigQuery/PostgreSQL 공식 문서**
- **핵심 강점**:
  - Query Plan 분석 방법 제시
  - Partitioning, Clustering 가이드
  - 실제 프로덕션 레벨 최적화 기법
- **한계**: 너무 기술적, 초보자 진입장벽 높음

### 5) **YouTube SQL 튜토리얼 (Alex the Analyst)**
- **핵심 강점**:
  - 무료, 친절한 설명
  - 포트폴리오 프로젝트 제작 가이드
  - 실제 데이터셋 활용
- **한계**: 최적화보다 기초 문법 중심

## 2. 사용자들이 공통적으로 불만을 느끼는 점

### 📚 **이론과 실습의 괴리**
- 온라인 강의는 "SELECT * FROM" 수준, 면접은 3-way JOIN + Window Function
- "어떻게 풀어야 하는가"는 배우지만 "왜 이 방식이 더 빠른가" 설명 부족

### ⏱️ **성능 최적화 경험 부족**
- 대부분의 튜토리얼은 "정답만 맞으면 OK"
- 실무에서 중요한 "이 쿼리는 10초 → 0.3초로 줄였습니다" 경험 불가능

### 🎯 **비즈니스 맥락 부재**
- "테이블 A와 B를 조인하세요" 같은 추상적 문제
- 실제 업무는 "이번 달 재구매율이 낮은 이유를 찾아라" 같은 비즈니스 질문

### 💰 **플랫폼 구독료 부담**
- LeetCode Premium $35/월, StrataScratch $49/월
- 취업 준비생에게는 부담스러운 가격

### 🔄 **피드백 부재**
- 혼자 공부하면 "이 쿼리가 최선인가?" 확신 불가
- 더 나은 방법이 있는지 알 수 없음

## 3. 아직 아무도 해결 못한 문제

### 🤖 **자동 쿼리 최적화 제안**
- **문제**: 사용자가 작성한 비효율적 쿼리를 분석해 "이렇게 고치세요" 제안하는 툴 부재
- **미해결 영역**:
  - Subquery를 CTE로 변환 제안
  - SELECT *를 필요한 컬럼만 선택하도록 리팩토링
  - 불필요한 JOIN 탐지

### 📊 **Before/After 성능 비교 자동화**
- **문제**: "최적화 전 3초 → 최적화 후 0.5초" 같은 정량적 증명 어려움
- **미해결 영역**:
  - Query Plan 자동 분석 및 시각화
  - Execution Time 벤치마크 자동 실행
  - 병목 구간(Bottleneck) 하이라이트

### 💡 **비즈니스 맥락이 있는 SQL 문제 생성**
- **문제**: 대부분의 연습 문제는 "employees 테이블에서..." 같은 추상적 설정
- **미해결 영역**:
  - "라프텔의 이번 달 신규 구독자 리텐션을 계산하세요" 같은 실무형 문제
  - 산업별(이커머스, SaaS, 핀테크) 특화 문제 세트

### 🎓 **면접 대비 "설명 연습" 도구**
- **문제**: 쿼리를 짜는 것과 "왜 이렇게 짰는가" 설명하는 것은 다른 스킬
- **미해결 영역**:
  - AI가 면접관 역할로 "이 CTE는 왜 썼나요?" 질문
  - 답변의 명확성/정확성 평가

## 4. 진입하기 좋은 틈새시장

### 🎯 **타겟 1: 취업 포트폴리오용 SQL 챌린지 컬렉션**
- **기회**: 
  - "30 Days of SQL" 같은 체계적 커리큘럼 부재
  - GitHub에 올릴 수 있는 "깔끔한 SQL 문제 풀이집" 수요
- **차별점**:
  - Day 1~30까지 난이도 순차 증가
  - 각 문제마다 Before/After 최적화 포함
  - README에 "이 문제에서 배운 것" 정리

### 🎯 **타겟 2: 데이터 분석가 면접 필수 30문제**
- **기회**:
  - "이 문제만 풀면 면접 70% 커버" 같은 필수 문제 리스트 부재
  - 에이블리, 토스, 라프텔 기출 유형 정리 필요
- **차별점**:
  - 실제 기업 JD 분석 → 출제 예상 문제 도출
  - 7일/30일 Rolling MAU, 재구매율 등 실무 지표 계산

### 🎯 **타겟 3: SQL 최적화 Before/After 쇼케이스**
- **기회**:
  - "이 쿼리를 10배 빠르게 만든 방법" 사례 모음 부족
- **차별점**:
  - 10개 최적화 사례 (각각 Before/After 성능 수치)
  - Execution Plan 스크린샷 포함
  - "면접에서 이렇게 설명하세요" 스크립트 제공
